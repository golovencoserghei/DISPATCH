# Ручные тесты и примеры

Требуют **реального браузера** с установленным и обновлённым расширением Dispatch
(нельзя гонять в CI). Перед запуском: расширение включено, мастер-тумблер вкл.

## stage3.mjs — прогон всех изменяющих инструментов
Поднимает локальную тест-страницу, сам открывает её через `open_tab` и прогоняет
`type`/`press_key`/`click`/`scroll`/`extract`/`debug_start`/`network`/`emulate`/
`navigate`/`close_tab` и full-page скрин. Давать доступ вручную не нужно.
```bash
node tests/manual/stage3.mjs
```

## playlist-export.mjs — пример сбора данных
Собирает названия треков открытого плейлиста Spotify (виртуализированный список:
скролл + накопление уникальных). Дай доступ вкладке страницы плейлиста.
```bash
node tests/manual/playlist-export.mjs
```
